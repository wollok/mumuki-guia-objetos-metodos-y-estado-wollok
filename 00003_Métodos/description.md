¿Cómo hacemos que un objeto entienda un mensaje? Tenemos que declararle un **método**: 

```scala
object pepita {
   method volarEnCirculos() {
   }
}
```

Un método contendrá la descripción de **qué hacer cuando se recibe un mensaje del mismo nombre**.
